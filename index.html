Command Registry

A fully extensible command engine with tab-completion, history navigation (↑↓), and real-time output streaming. Register custom commands in seconds.

Extensible
🔑
Whitelist Key System

Account-locked key authentication. Keys are bound per-username and auto-saved locally — never type your key twice. Paid commands stay protected.

Secure
🎮
Multi-Game Support

Dedicated command sets for Creatures of Sonaria, Blood Zone, Dog Pound and more. Use game cos to list all available commands per game.

Auto-Resume

Loop commands (autofarm, killall, etc.) are saved to disk on execution. Rejoin a server and Poseidon will prompt — or auto-restart — exactly where you left off.

Persistent

Live overhead labels for every player — species, tier, health bar, bite cooldown, sprint speed, ailments, and threat scoring. Sort by threat or distance.
